## Project Overview

This project analyzes customer data to understand factors that influence bike purchasing behavior.  
An interactive Excel dashboard was created to help stakeholders explore trends and patterns easily.



## Business Questions Answered

### 1. Average Income per Purchase
- Compared average income of customers who purchased a bike vs those who did not
- Segmented by gender (Male vs Female)
- Helps understand whether income level influences bike purchases

### 2. Commute Distance vs Bike Purchase
- Analyzed how far customers live from their workplace
- Compared bike purchases across different commute distance ranges
- Helps determine whether shorter or longer commutes impact buying decisions

### 3. Customer Age Bracket Analysis
- Grouped customers into age categories (Adolescent, Middle-Aged, Old)
- Identified which age group is most likely to purchase a bike
- Supports targeted marketing and customer segmentation



## Data Cleaning & Preparation

- Removed duplicate records
- Standardized categorical values (Gender, Marital Status)
- Created age brackets using IF formulas
- Verified data consistency using filters



## Dashboard Features

- Interactive Pivot Tables & Pivot Charts
- Slicers for:
- Marital Status
- Education
- Region
- Enables dynamic exploration of customer behavior



## Tools Used

- Microsoft Excel
- Pivot Tables & Pivot Charts
- IF formulas
- Data Cleaning & Transformation
- Dashboard Design



## Key Insights

- Income and gender show clear patterns in bike purchasing behavior
- Customers with certain commute distances are more likely to buy bikes
- Middle-aged customers represent the largest purchasing group
